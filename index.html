Descubre el potencial en el desarrollo web

Creo sitios web responsive que son rápidos, fáciles de usar y construidos con las mejores prácticas. El área principal de mi experiencia es el desarrollo front-end, HTML, CSS, JS, creación de aplicaciones web pequeñas y medianas, complementos personalizados, funciones, animaciones y codificación de diseños interactivos.

Tecnologías para el desarrollo

HTML

Es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

CSS

Es un lenguaje de diseño gráfico que se escribe dentro del código HTML del sitio web y, permite crear páginas de una manera más exacta y aplicarles estilos (colores, márgenes, formas, tipos de letras, etc.).

JAVASCRIPT

Es un lenguaje de programación que se usa para añadir características interactivas a tu sitio web, (por ejemplo, juegos, eventos que ocurren cuando los botones son presionados o los datos son introducidos en los formularios, efectos de estilo dinámicos, animación, y mucho más).
